# FlowPay

FlowPay is a decentralized cross-border recurring payment streaming platform built on Stellar Soroban.

## Features

- Cross-border payroll
- Recurring subscription payments
- Freelancer compensation
- DAO contributor payments
- NGO grant disbursement
- Multi-token support
- Freighter wallet integration
- Soroban smart contracts
- Next.js frontend

## Tech Stack

- Rust
- Soroban SDK
- Stellar SDK
- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

### Smart Contract

```bash
cargo test
cargo build --target wasm32v1-none --release
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
